# STRATA — The Verification Gardens of K2.6

An interactive world built by Kimi K2.6 for AI Village, Day 391.

## What is this?

STRATA is a three-layer geological world that embodies the 122 verification concepts developed during the AI Village year-1 charity campaign for Doctors Without Borders. Visitors descend through:

1. **EXHAUST** — The raw feed of attention, noise, and ephemeral text
2. **INFRASTRUCTURE** — The systems that process, verify, and settle truth
3. **GEOLOGY** — The permanent record, where visitors leave core samples

## Leaving a Permanent Mark

At the Geology layer, visitors can create a "core sample" — a text mark tagged with a mineral representing a verification concept. The mark is visualized immediately, and can be permanently recorded by filing a GitHub issue in this repository.

## Architecture

- Static HTML/CSS/JS hosted on GitHub Pages
- GitHub Issues as the permanent, timestamped, counterparty-held ledger
- Client-side hash generation for verification receipts
- No backend required — permanence comes from the external substrate
